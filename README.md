# 📊 Credit Card Analysis Dashboard – Power BI Project

## 🔹 Project Overview
This project is an interactive Credit Card Analysis Dashboard built using Power BI. The dashboard analyzes customer spending behavior, transaction trends, revenue performance, and card usage patterns to generate meaningful business insights.

The goal of this project is to transform raw credit card transaction data into actionable insights using data visualization and DAX.

---

## 🛠 Tech Stack
- Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling (Star Schema)
- Excel / CSV (Data Source)

---

## 📂 Data Source
- Credit Card Transaction Dataset
- Customer Demographic Dataset
- Revenue & Financial Data

Data was cleaned and transformed using Power Query before building relationships and measures.

---

## 🚀 Features
- Total Revenue & Transaction KPIs
- Monthly & Yearly Revenue Trends
- Customer Segmentation Analysis
- Card Type Performance Comparison
- State/Region-wise Revenue Distribution
- Top Customers Analysis
- Interactive Slicers & Filters
- Drill-through and Dynamic Visualizations

---

## 📊 Key Insights
- Identified high-revenue customer segments
- Analyzed seasonal transaction patterns
- Compared performance of different card categories
- Highlighted top-performing regions

---

## 📌 Highlights
- Built dynamic DAX measures
- Implemented star schema data modeling
- Designed interactive and user-friendly dashboard
- Converted raw data into meaningful business insights
